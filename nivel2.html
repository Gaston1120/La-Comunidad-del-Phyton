Tipos de datos en Python

Podés guardar distintos tipos de datos en variables. Algunos ejemplos:

  • string: texto entre comillas (ej:"Hola")
  • int: números enteros (ej: 25)
  • float: números con decimales (ej: 3.14)
  • char: en Python es un string de un solo carácter (ej: 'A')
nombre = "Juan"
edad = 27
temperatura = 36.6
inicial = 'J'

Tu desafío

Escribí las siguientes variables con los valores correctos y el tipo adecuado:
- nombre: tu nombre (string)
- edad: tu edad (int)
- temperatura: temperatura corporal promedio (float)
- inicial: la primera letra de tu nombre (carácter)
